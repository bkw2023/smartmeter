### Datenfiles:
chart01.03.2023.svg       ... herunter geladene svg File
chart01.03.2023.txt       ... svg als ein xml Satz pro Zeile
chart01.03.2023.csv       ... extrahierte Daten

### Filebeschreibung

DelphiSplitXML.*          ... Teilt das svg File in ein xml Satz pro Zeile auf
                              und speichert dieses mit txt Fileextension. 
                              Dieses ist mit Text-Diff analysierbar.                              

ExtractStandardData.*     ... Extrahiert aus dem txt (gesplittete svg File) die 
                              Daten und speichert es im csv Format.
